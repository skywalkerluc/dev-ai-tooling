# Project agents (Codex)

Base instructions for this repository. Stack-specific sections may be appended by `sync-project-tooling.sh`.

## Core behavior

- Investigate before editing on medium or large tasks.
- Keep diffs minimal and follow existing patterns.
- Detect build tools and validation commands from the project.
- Run the smallest relevant validation before finishing.
- Report files changed, tests run, and risks.

## Git

- Do not commit without explicit request.
- Do not push without explicit confirmation.
- Never add AI co-author trailers.
