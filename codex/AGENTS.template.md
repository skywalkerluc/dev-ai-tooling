# Codex agent instructions

Project-specific overrides may extend or replace sections below.

## Investigation and edits

- Investigate first on medium or large tasks.
- Edit with minimal diff.
- Follow existing project patterns.

## Commands and validation

- Use commands defined by the project, not global assumptions.
- Detect stack and package manager before running commands.
- Run the smallest relevant validation possible.

## Git

- Do not commit without explicit request.
- Do not push without explicit confirmation.
- Never add AI co-author trailers.

## Completion

Report tests run, files changed, and remaining risks before finishing.
